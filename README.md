# 🍫 Power BI Data Analysis Project – Australian Chocolate Factory

## 📌 Overview
This project presents an interactive **Power BI dashboard** developed to analyze sales and production data of an Australian chocolate manufacturing company.

The dashboard focuses on converting raw data into meaningful insights to support **data-driven decision-making**, improve operational visibility, and track business performance effectively.

---

## 🎯 Objectives
- Analyze sales performance across products and regions  
- Monitor production trends and efficiency  
- Track key performance indicators (KPIs)  
- Enable interactive data exploration for stakeholders  

---

## 🛠️ Tools & Technologies Used
- Power BI Desktop  
- DAX (Data Analysis Expressions)  
- Power Query Editor  
- Data Modeling (Star Schema Approach)  

---

## ⚙️ Data Processing & Modeling
- Performed data cleaning and transformation using Power Query  
- Built relationships between multiple tables  
- Implemented a star schema model for optimized querying and performance  

---

## 📊 Dashboard Features
- Interactive filters and slicers  
- KPI indicators for performance tracking  
- Sales trend analysis  
- Region-wise insights  
- Product-level performance metrics  

---

## 🔢 Key DAX Calculations
- Total Sales  
- Profit / Revenue Metrics  
- Growth Trends  
- Aggregated KPIs  

---

## 💡 Business Insights (Expected Outcomes)
- Identification of high-performing products  
- Detection of seasonal trends  
- Insights into regional sales variations  
- Understanding production and sales alignment  

---

## 📁 Project File
- Power BI Report File: `PowerBI_Report.pbix`  

> ⚠️ Note: The dataset used in this project is embedded within the `.pbix` file and not shared separately.

---

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open it using Power BI Desktop  
3. Explore the dashboard using filters and visuals
